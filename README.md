# volitve-2026

Zbirka programov političnih strank, povzetkov in analiz za parlamentarne volitve v Sloveniji, **22. marca 2026**.

Repozitorij vsebuje programske dokumente strank, njihove povzetke, ankete in tematske analize – pripravljeno za informirano odločanje pred volitvami. Vsi povzetki in analize so napisani nepristransko in brez vrednostnih sodb.
